# Meteor LRPT Proc
Process images from Meteor M2 LRPT.
## Setup
```bash
pip3 install Pillow
```

## Supported processors
|Name|Status|
|----|------|
|Veg        🌳| Stable|
|Rainfall   🌧| Stable|
|No Veg     🏜| Stable|
